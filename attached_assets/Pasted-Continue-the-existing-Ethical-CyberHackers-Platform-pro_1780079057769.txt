Milestone 25D — Split-Screen Mission Control Layout

Purpose:
Keep the Cyber Missions Map visible during active investigations so the student always feels inside a mission-control environment. Reduce terminal-screen clutter by moving instructions, progress, manager messages, and mission context into a right-side Mission Control panel.

Do not remove terminal access.
Do not break Mission 1 or Mission 2.
Do not remove existing mission logic.
Do not add backend, AI, authentication, or new pages.

Layout Goal During Active Mission:

LEFT SIDE:

* Terminal
* Command/action buttons
* Investigation interactions

RIGHT SIDE:

* Compact Mission Map
* Current Objective
* Manager Transmission
* Alert Status
* Evidence Board
* Threat Level / Trust Score
* Mission Progress

---

1. ACTIVE MISSION SPLIT LAYOUT

---

When a mission is launched, use a two-column layout:

Left column:
70% width

Right column:
30% width

Left column contains:

* terminal
* command buttons
* investigation board interactions if needed

Right column contains:

* compact mission map
* current objective
* manager chat/transmission
* alert/threat status
* evidence summary
* progress checklist

On smaller screens:
stack vertically with terminal first.

---

2. COMPACT MISSION MAP

---

Create a compact version of the Cyber Missions Map for the right side.

It should show:

* Mission 1 node
* Mission 2 node
* Mission 3 locked node
* path lines
* current mission highlighted
* completed mission checkmarks
* locked mission dim state

This should be smaller than the full map but visually consistent.

Label:
Mission Route

---

3. MOVE INSTRUCTIONS OUT OF TERMINAL AREA

---

Remove or minimize repeated instruction blocks near the command area.

Keep only:

* Current Objective
* Hint button
* Command/action buttons

Detailed instructions should appear in the right-side Mission Control panel.

---

4. CURRENT OBJECTIVE IN RIGHT PANEL

---

Add a high-visibility card:

Current Objective

* Open the documents folder.

* Submit your finding.
* Complete the quiz.

---

Move manager guidance into the right panel.

* manager avatar initials
