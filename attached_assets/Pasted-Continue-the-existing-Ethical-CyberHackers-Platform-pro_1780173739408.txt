3. MISSION 2 GUIDED NETWORK PACING

---

Do not show all network commands at once.

Reveal one step at a time:

Step 1:
ip addr

Student identifies local network position.

Step 2:
ping 10.0.0.8

Shows unreachable host.

Student interprets it as not currently actionable.

Step 3:
ping 10.0.0.5

Shows reachable host.

Student interprets it as active target.

Step 4:
nmap 10.0.0.5

Shows open services.

Step 5:
review services

Student interprets exposed services.

Each step should:

* show only the current required action
* update Current Objective
* update manager response
* update evidence/analyst confidence when appropriate

---

4. MISSION 2 REASONING PROMPTS

---

After each major output, ask a short reasoning question.

Examples:

After ip addr:
“What does this tell you?”
Correct:
“This identifies the student workstation’s local network address.”

After ping 10.0.0.8:
“What does this result suggest?”
Correct:
“The host is not reachable right now.”

After ping 10.0.0.5:
“What does this result suggest?”
Correct:
“The host is reachable and can be investigated further.”

After nmap:
“What is the main security concern?”
Correct:
“The host exposes network services that should be reviewed.”

After review services:
“What should Blue Team do next?”
Correct:
“Recommend security review of exposed services.”

7. MISSION 2 BLUE TEAM DECISION MOMENT

---

After open services are discovered and interpreted, show:

BLUE TEAM RESPONSE REQUIRED

Text:
“A reachable host is exposing SSH, HTTP, and HTTPS. What should Blue Team do next?”

Options:
A. Recommend security review of exposed services
B. Ignore the services because open ports are always safe
C. Shut down the entire network immediately
D. Continue scanning unrelated hosts

Correct/best:
A. Recommend security review of exposed services

Consequences:

A:

* Trust increases
* Threat level stabilizes
* Containment progress increases
* Manager:
  “Good recommendation. Exposed services should be reviewed before attackers can abuse misconfigurations.”

B:

* Trust decreases slightly
* Threat level remains elevated
* Manager:
  “Open services should not be ignored. They define attack surface.”

C:

* Manager warns this is excessive
* Trust does not increase
* Threat level stabilizes only slightly
* Message:
  “Containment should be proportional to evidence.”

D:

* Incident pressure rises slightly
* Manager:
  “Stay focused on the active target before expanding scope.”
