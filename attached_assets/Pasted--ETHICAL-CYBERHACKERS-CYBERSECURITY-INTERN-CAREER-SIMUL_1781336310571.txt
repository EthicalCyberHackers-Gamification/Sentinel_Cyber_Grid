# ANALYST NOTEBOOK REFACTOR

Current notebook reveals too much.

Replace with investigation-driven discovery.

Sections:

FACTS

HYPOTHESES

------------------------------------------------------------------

# FACTS

Only verified information.

Examples:

✓ Contractor account accessed restricted files

✓ Unknown device discovered

✓ Login succeeded after repeated failures

Facts are earned.

Not pre-populated.

------------------------------------------------------------------

# HYPOTHESES

Potential explanations.

Examples:

□ Password guessing attack

□ Contractor account compromise

□ Unauthorized access

Hypotheses change as evidence accumulates.

------------------------------------------------------------------

# UNKNOWNS

Outstanding investigation questions.

Examples:

□ How was access obtained?

□ Is the device authorized?

□ Was data transferred externally?

Investigation should reduce unknowns.

------------------------------------------------------------------

# RECOMMENDATIONS

Student-generated recommendations.

Examples:

□ Restrict access

□ Escalate

□ Recommend MFA

□ Legal review

□ Forensic investigation
